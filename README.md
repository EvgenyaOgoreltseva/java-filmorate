# FilmoRate
Бэкенд для сервиса, который будет работать с фильмами и оценками пользователей, а также возвращать топ-5 фильмов, рекомендованных к просмотру. 

Стек : Java + Spring Boot + Maven + Lombok + JUnit + RESTful API
